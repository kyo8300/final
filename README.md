## 喵喵對對碰
它是個撲克牌遊戲。 翻開兩張圖片相同的就算配對成功，全部配對完成就算通過遊戲。

## 遊戲玩法
使用者先點擊一張撲克牌，接下來點擊另外一張撲克牌。
如果兩張圖片相同就算配對成功，以此類推。
最後會有分數。
不過如果時間超過就算Game Over。

## 使用的技術
- HTML
  - ul, id
- CSS
  - position, float
- Javascript(JQuery)
  - Array, Function,animate(), random(), addClass(), removeClass()
  
## 特點
- 撲克牌排的很整齊。有美感。
- 撲克牌翻面的設計也相當美觀。
- 有計時以及分數。
